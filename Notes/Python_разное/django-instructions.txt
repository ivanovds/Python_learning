Когда создаешь в Джанго модель(таблицу), то чтоб это отобразилось в локальной БД юзаешь команды: makemigrations отвечает за сохранение состояния моделей в файле миграции - аналог коммита -
 а migrate отвечает за их применение к базе данных.

1) python manage.py makemigrations myapp - команда для создания миграций в апликэйшене(папке) "myapp"
2) python manage.py migrate - внесение изменений в БД
